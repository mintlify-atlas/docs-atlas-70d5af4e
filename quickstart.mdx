This guide walks you through creating a diagram from scratch and exporting it. The entire workflow takes about five minutes.

## Before you begin

Make sure you have draw.io Desktop installed. If not, follow the [installation guide](/installation).

## Create your first diagram

<Steps>
  <Step title="Open draw.io Desktop">
    Launch draw.io Desktop from your Applications folder (macOS), Start menu (Windows), or application launcher (Linux). A new blank diagram opens automatically.
  </Step>
  <Step title="Add shapes">
    Click and drag any shape from the **shape panel** on the left onto the canvas. You can search for specific shapes using the search box at the top of the panel.

    To add a shape quickly, double-click an empty area of the canvas. A shape picker appears where you can type to filter shapes.
  </Step>
  <Step title="Connect shapes">
    Hover over a shape until you see blue connection arrows appear on its edges. Click and drag from one of these arrows to another shape to create a connector.

    <Tip>
      Hold **Shift** while dragging from a connection point to create a floating (non-anchored) connector.
    </Tip>
  </Step>
  <Step title="Add labels">
    Double-click any shape or connector to edit its label. Press **Escape** or click outside to finish editing.
  </Step>
  <Step title="Save your diagram">
    Press **Cmd+S** (macOS) or **Ctrl+S** (Windows/Linux) to save. Choose a location and filename. draw.io saves files in the `.drawio` format, which is XML-based and human-readable.

    <Note>
      Your diagram is saved entirely on your local machine. No data is sent to any server.
    </Note>
  </Step>
</Steps>

## Export your diagram

Once your diagram is ready, export it for sharing or embedding.

<Steps>
  <Step title="Open the export dialog">
    Go to **File > Export As** and choose your desired format.
  </Step>
  <Step title="Choose a format">
    | Format | Best for |
    |--------|----------|
    | **PDF** | Print, documents, sharing |
    | **PNG** | Presentations, web, screenshots |
    | **SVG** | Scalable graphics, web embedding |
    | **HTML** | Interactive diagrams on the web |
    | **XML** | Backup, version control, round-tripping |

  </Step>
  <Step title="Configure export options">
    Each format has its own options — for example, PNG lets you set scale and enable transparent background. Configure as needed, then click **Export**.
  </Step>
</Steps>

## Open an existing file

To open an existing `.drawio` or `.vsdx` file:

- **Drag and drop** the file onto the draw.io Desktop window
- Use **File > Open** and browse to the file
- **Double-click** the file in Finder (macOS), File Explorer (Windows), or your file manager (Linux)

<Tip>
  On macOS, you can press **Space** in Finder to preview a `.drawio` file using Quick Look without opening the app.
</Tip>
