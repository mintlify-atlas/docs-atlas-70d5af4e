draw.io Desktop is a free desktop application for creating flowcharts, UML diagrams, network diagrams, org charts, and more. It is built on [Electron](https://electronjs.org/) and wraps the [core draw.io editor](https://github.com/jgraph/drawio) as a git submodule. All diagram data stays on your machine — nothing is transmitted to external servers.

  </Card>
  <Card title="Plugin system" icon="puzzle-piece">
    Load custom plugins from your local machine. Plugins are disabled by default and must be opted in.
  </Card>

## Architecture

draw.io Desktop is an Electron application. The main process (`src/main/electron.js`) handles window management, file I/O, menus, auto-update, and CLI export. The renderer process loads the draw.io web application from a local file path — no web server is involved.

A preload script (`src/main/electron-preload.js`) bridges the renderer and main process using Electron's `contextBridge`, exposing only a narrow, validated API surface. All IPC calls are validated to ensure they originate from the local draw.io code.

The draw.io editor itself lives in a `drawio/` git submodule, which is the same codebase that powers the web application at draw.io.

## What draw.io Desktop is not

<Warning>
  draw.io Desktop does not accept external contributions. Pull requests are not reviewed or merged. The project is maintained exclusively by JGraph.
</Warning>

<Info>
  No diagram data, usage analytics, or telemetry is ever sent to external servers. A Content Security Policy is enforced on the renderer to make external transmission impossible, even accidentally.
</Info>
